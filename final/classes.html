drawing and painting

in this class we are focusing on basic elements of drawing and painting skills such as perspective art, watercolor, inking, etc. we recently ended with our final project which was to draw a manga face portrait, ink it, watercolor it, and finally, digitize the plain inked version.

algebra 2

in this class we've been working a lot with polynomials. we started this trimester off with a lot more linear functions but then progressed into polynomials which we have continued up until now. for our final project we had a 'mathematical thinking project.' this was a project in which we were given 3 problems. in partners, we chose 2 of these problems to solve. through the whole problem solving process, we had to note our work and steps and in the end, create a write-up. we also got to choose one of the two problems to present on.

spanish 2

this trimester, we've been working a lot more with preterite. we've worked with both regular and irregular verbs and have learned a lot of new vocabulary, specifically related to sports and daily routines. for our final, we had a oral project. in partners, you could pick any topic to present on for 8-10 minutes total. you had certain requirements that you had to fulfill. i partnered with maya and we talked about different sports players in the past tense.

study hall

in this class... i've been studying... its study hall... not much more explanation needed. if you need for information... use this link to learn more

WEB

in this class we've been learning how to use different aspects of code to make a webpage. we've learned html, css, and js. we've been coding in visual studio code. we've had many demos (access them here) and for our final we are using all the skills we've learned to make a webpage about ourselves.

scientific thinking 3

this trimester, we have been learning about astronomy and recently transitioned to learning about light, sound, and waves. we've had a few tests and labs. for example we recently did a lab called 'color addition lab' in which we experimented with color filters and light to create different combinations of colors. for our final, we have a comp ook check which is where we submit our comp book which we have done all our labs in. for this comp book check, we only have 2 labs and it's not weighted the same as a final, it's similar to an MA.

historical thinking 3

in this class we are learning about religions, currently in our judaism unit. this year we have learned about: hinduism, sikhism, buddhism, and judaism. we've been connecting historical events and figures to these religions, for example, gandhi to buddism, tiber and china's dispute to buddhism, and currently, the holocaust to judaism. we also go to visit places of worship for these religions to help us understand them more. currently we have only visited a gurudwara (sikhism place of worship). in spring tri we will be learning about islam and christianity and having a lot more field trips. we also didn't really have a final for this class.

literary thinking 3

in this class we started the trimester out with reading the hunger games, we had TELA discussions and class analysis about this book, focusing on themes of: survival, dystopias, and hero's journey. we've focused a lot of the hero's journey even through the fall trimester. right now we are in a unit called 'litshop' where we pick a book and write a letter to someone about it for our final.
